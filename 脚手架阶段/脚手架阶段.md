## 安装Vue-CLI

### Vue-CLI 3

```
npm install -g @vue/cli
```

### Vue-CLI 2

安装脚手架 2 版本只需要在安装 3 之后拉取 2 的模板

```
npm install -g @vue/cli
npm install -g @vue/cli-init
```

## 创建项目

### Vue-CLI 2

**初始：**

```
//vue init <template-name> <project-name>
vue init webpack vuecli2test
```

