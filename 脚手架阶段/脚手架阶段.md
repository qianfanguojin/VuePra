## 安装Vue-CLI

### Vue-CLI 3

```
npm install -g @vue/cli
```

### Vue-CLI 2

安装脚手架 2 版本只需要在安装 3 之后拉取 2 的模板

```
npm install -g @vue/cli
npm install -g @vue/cli-init
```
